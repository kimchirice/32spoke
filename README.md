## 32spoke
a webapp helps wheelbuilder to calculate spoke length for the coming build


## wheel variables
  1. Hub
    * Hub name
    * Left flange diameter
    * Right flange diameter
    * Left flange to center
    * Right flange to center 
    * Spoke hole diameter
    * Hub weight
  2. Rim
    - Rim name
    - Rim ERD
    - Offset spoke bed(OSB)
    - Max rim tension
    - Rim weight
